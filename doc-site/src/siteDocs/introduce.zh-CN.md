# BsVue
`Bootstrap Vue` 简称 `BsVue`，它是基于Vue.js 3.x及Bootstrap UI的一套组件库，

它适用于面向用户的或后台管理类的产品，适配PC、平板、手机等不同类型的设备。

## 特性
- 开箱即用的高质量 Vue 组件
- 适配PC、平板、手机等不同类型的设备
- 友好的 API ，自由灵活地使用空间

## 支持环境

- 适用于 Vue.js 3，兼容现代浏览器，不支持 IE
