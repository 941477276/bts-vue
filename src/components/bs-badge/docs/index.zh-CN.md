徽章（badge）是一种小型的用于计数和打标签的组件

## API

### Badge 属性

| 参数       | 说明              | 类型                                                                                | 默认值        |
|----------|-----------------|-----------------------------------------------------------------------------------|------------|
| tag      | 渲染dom都标签名称      | string                                                                            | span       |
| type     | 颜色类型            | `primary`/ `secondary`/ `success`/ `danger`/ `warning`/ `info`/ `light`/ `dark`   | primary    |
| pill     | 是否为胶囊样式         | boolean                                                | false      |
| circle   | 是否为小圆点样式        | boolean                                               | false      |
